## About

<img src="https://raw.githubusercontent.com/siddharth018/Laravel8-Shopping-Cart/main/public/shoppingcart.png" width="100%">

First Step: download in your system.

git clone https://github.com/siddharth018/Laravel8-Shopping-Cart.git

    cd Laravel8-Shopping-Cart

composer install

    cp .env.example .env

Put your database credentils .env file.

    DB_CONNECTION=mysql
    DB_HOST=127.0.0.1
    DB_PORT=3306
    DB_DATABASE=laravel_cart
    DB_USERNAME=root
    DB_PASSWORD=root@123

Run Migration command

    php artisan migrate

Run ProductSeeder Seeder command
    
    php artisan db:seed --class=ProductSeeder

Step second: Run server

    php artisan serve

check user logged in
